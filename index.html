This page contains information and links to some of my work.

Most of my personal projects gravitate towards GIS in some manner. This is because I have worked in this area and having a meaningful map as an end product I find quite interesting.

Data is really the key so I am often on the hunt for a new and/or better source. I do use 2010 census data in some places which is dated but free. For real world production purposes, there are 3rd party vendors out there that sell updated/estimated data.


Hardware and Application Deployment:

All deployments are to Raspberry Pi's or S3 for static websites.


Raspberry Pi 2 B+

- Raspbian OS (Jessie)

- Postgres 9.4 with Postgis 2.4 - DB contains a number of geospatial datasets


Raspberry Pi 3 B

- Hypriot OS (Debian v9.4)

- Docker Swarm: Global Administrative Boundaries in MongoDB

- Docker App Containers: CoordinateInfo-API, GO API's (various), Mappy-Data, Shake-Shake

Mappy-Data

Description: Given a polygon on the map, calculate demographics within such polygon.
Tech Stack: Ruby on Rails, JS, PostgreSQL/PostGIS, Route360.net, Google Maps API, Census Bureau Data, Docker
Deployment: Raspberry Pi - http://www.zotac1.ddns.net:3300
GitHub: https://github.com/greghorne/Mappy-Data

Go API (Intersect USA)

Description:Given an x,y determine if it intersects U.S. State boundaries. This will be used in a different project.

CoordinateInfo-Client

Description:A basic mapping client to consume data from CoordinateInfo-API.

CoordinateInfo-API

Description: Rails API, given an x,y coordinate, return JSON with municipality information.
Tech Stack: Rails 5 API, Ruby, Postman, PostgreSQL/PostGIS, MongoDB Docker Swarm
Deployment: Raspberry Pi - http://zotac1.ddns.net:3100/api/v1/coord_info?longitude_x=88.445670&latitude_y=23.243660
GitHub: https://github.com/greghorne/CoordinateInfo-API

Shake-Shake

Description: Display a map with information on the latest earthquake via data provided by the USGS. This webapp is an exercise in using Server-Sent Events.
(note: server-sent events are not supported by IE/Edge as of this writing)
Tech Stack: Ruby, JS, Sinatra, Server-Sent Events, Leaflet, OpenStreetMaps, USGS REST API Server, Docker
Deployment: Raspberry Pi - http://zotac1.ddns.net:3200
GitHub: https://github.com/greghorne/Shake-Shake

Load Census Blocks

Description: Shell script used to retrieve Census Block data and load it into a PostgreSQL/PostGIS database.
Tech Stack: Bash shell scriptting (OS X), Census Bureau FTP site, PostgreSQL/PostGIS
Deployment: DB deployed onto a Raspberry Pi 2
GitHub: https://github.com/greghorne/loadCensusBlocks
