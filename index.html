Grand Junction Tulsa

Description: Truck accessories shop that only wanted a single page website (web presence only).

GitHub: https://github.com/greghorne/grandjunction2.git


MappyData

Description: Given a polygon on the map, calculate demographics within such polygon.

Tools Used: Ruby on Rails, JS, PostgreSQL/PostGIS, Route360.net, Google Maps API, Raspberry Pi 2

Deployment: Heroku

GitHub: https://github.com/greghorne/mappy


Load Census Blocks

Description: Shell script used to retrieve Census Block data and load it into a PostgreSQL/PostGIS database.

Tools Used: Bash shell scriptting (OS X), Census Bureau FTP site, PostgreSQL/PostGIS

Deployment: DB deployed onto a Raspberry Pi 2 and used in MappyData project

GitHub: https://github.com/greghorne/mappy
