This page contains information and links to some of my work.

One may notice that a lot of my personal projects gravitate towards GIS in some manner. This is because I have worked in this area and having a meaningful map as an end product I find quite interesting.

Data is really the key so I am often on the hunt for a new and/or better source. I use 2010 census data in some places which is dated but free. For real world production purposes, there are 3rd party vendors out there that sell updated/estimated data.


Hardware and Application Deployment:

Mappy-Data may take a moment to spin up on a Heroku free tier.

Originally, I had deployed my work onto Heroku using their free tier. To prevent my apps from going to sleep on Heroku and to learn more about Docker, I have been moving my apps to my own local devices which are Raspberry Pi's.

Currently, only Mappy-Data is still deployed to Heroku and static webpages are deployed to S3.


Raspberry Pi 2 B+

- Raspbian OS

- Postgres 9.4 with Postgis 2.4 - DB contains a number of geospatial datasets


Raspberry Pi 3 B

- Hypriot OS (Debian v9.4)

- All deployments using Docker

- Docker Swarm: Global Administrative Boundaries in MongoDB

- Docker Containers: CoordinateInfo-API, go-api-intersect-usa, go-api-targomo, Shake-Shake
