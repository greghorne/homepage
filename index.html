MappyData

Description: Given a polygon on the map, calculate demographics within such polygon.

Tools/Resources: Ruby on Rails, JS, PostgreSQL/PostGIS, Route360.net, Google Maps API, Raspberry Pi 2, Census Bureau Data

Deployment: Heroku

GitHub: https://github.com/greghorne/mappy


Load Census Blocks

Description: Shell script used to retrieve Census Block data and load it into a PostgreSQL/PostGIS database.

Tools/Resources: Bash shell scriptting (OS X), Census Bureau FTP site, PostgreSQL/PostGIS

Deployment: DB deployed onto a Raspberry Pi 2 and used in MappyData project

GitHub: https://github.com/greghorne/mappy
