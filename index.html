Push Sinatra

Description:

 Display a map with information on the latest earthquake via data provided by the USGS. This webapp is an exercise in using Server-Sent Events. (note: server-sent events are not supported by IE/Edge as of this writing)

Tech Stack:

 Ruby, JS, Sinatra, Server-Sent Events, Leaflet, OpenStreetMaps, USGS REST API Server

Deployment:

Heroku - https://shake-shake.herokuapp.com

GitHub:

 https://github.com/greghorne/push_sinatra


MappyData

Description:

 Given a polygon on the map, calculate demographics within such polygon.

Tech Stack:

 Ruby on Rails, JS, PostgreSQL/PostGIS on a Raspberry Pi 2, Route360.net, Google Maps API, Census Bureau Data

Deployment:

Heroku & Raspberry Pi II - http://www.mappydata.com

GitHub:

 https://github.com/greghorne/mappy


Load Census Blocks

Description:

 Shell script used to retrieve Census Block data and load it into a PostgreSQL/PostGIS database.

Tech Stack:

 Bash shell scriptting (OS X), Census Bureau FTP site, PostgreSQL/PostGIS, Raspberry Pi II

Deployment:

DB deployed onto a Raspberry Pi 2

GitHub:

 https://github.com/greghorne/loadCensusBlocks
